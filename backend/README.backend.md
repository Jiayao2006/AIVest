# Backend (Express API)

Run:
```
cd backend
npm install
npm start
```

Dev with autoreload:
```
npm run dev
```

Endpoints:
- GET /api/health
- GET /api/clients
- GET /api/clients/search?q=term
- GET /api/clients/:id

Environment:
- PORT (default 5000)

Future enhancements: auth, persistence, pagination, filters.
